## PHẦN 2: HƯỚNG DẪN SEO

### 2.1 Cấu trúc URL tối ưu

```
✅ Tốt:
https://cic-platform.vn/
https://cic-platform.vn/edt-core
https://cic-platform.vn/news/1

❌ Tránh:
https://cic-platform.vn/?page=1&id=abc
```

### 2.2 Meta Tags chuẩn SEO

#### Title Tags (60-70 ký tự):
```
Trang chủ: "CIC Platform | Nền tảng Quản trị số & BIM hàng đầu Việt Nam"
EDT-Core: "CIC Platform - Hệ điều hành số ngành Xây dựng"
Tin tức: "[Tiêu đề bài] | CIC Platform"
```

#### Meta Description (150-160 ký tự):
```
Trang chủ: "Trung tâm Kiến thức và Dịch vụ BIM & Digital Twin hàng đầu tại Việt Nam. Cung cấp giải pháp, đào tạo và tư vấn chuyển đổi số ngành xây dựng."
```

### 2.3 Keywords chiến lược

#### Primary Keywords:
- BIM Việt Nam
- Quản lý dự án xây dựng
- Chuyển đổi số xây dựng
- CDE nền tảng
- Digital Twin xây dựng

#### Secondary Keywords:
- NĐ 175/2024
- NĐ 111/2024
- ISO 19650
- Tiêu chuẩn BIM TCVN
- Phần mềm quản lý công trình

#### Long-tail Keywords:
- "Phần mềm quản lý dự án xây dựng tốt nhất"
- "Hướng dẫn áp dụng BIM theo NĐ 175"
- "Giải pháp CDE cho doanh nghiệp nhỏ"
- "Đào tạo BIM nội bộ doanh nghiệp"

### 2.4 Heading Structure

```html
<h1>Chỉ 1 H1 mỗi trang - Tiêu đề chính</h1>
  <h2>Section 1</h2>
    <h3>Subsection 1.1</h3>
    <h3>Subsection 1.2</h3>
  <h2>Section 2</h2>
```

### 2.5 Internal Linking Strategy

```
Trang chủ ──► Services ──► EDT-Core
    │              │
    ▼              ▼
  Tools ◄──── Pricing ──► Contact
    │
```

## PHẦN 3: TÍNH NĂNG MARKETING TRÊN WEBSITE

- Trả lời FAQ tự động

## PHẦN 4: CONTENT MARKETING STRATEGY

### 4.1 Loại nội dung hiệu quả

| Loại nội dung | Mục tiêu | Tần suất |
|---------------|----------|----------|
| Tin pháp lý | Authority, SEO | Khi có NĐ/TT mới |
| Hướng dẫn BIM | Lead generation | 2 bài/tuần |
| Case study | Trust building | 1/tháng |
| Video tutorial | Engagement | 1/tuần |
| Infographic | Social sharing | 2/tháng |

### 4.2 Content Calendar mẫu

**Tuần 1:**
- Thứ 2: Tin pháp lý mới
- Thứ 4: Bài hướng dẫn kỹ thuật
- Thứ 6: Tips & Tricks ngắn

**Tuần 2:**
- Thứ 2: Case study khách hàng
- Thứ 4: So sánh/Review tool
- Thứ 6: Q&A từ cộng đồng

### 4.3 Tối ưu bài viết tin tức

```markdown
## Checklist SEO cho bài viết:

☐ Title chứa keyword chính (60-70 ký tự)
☐ Meta description hấp dẫn (150-160 ký tự)
☐ URL ngắn gọn, có keyword
☐ H1 duy nhất, H2-H3 phân cấp rõ ràng
☐ Ảnh có alt text mô tả
☐ Internal links đến bài liên quan
☐ CTA rõ ràng cuối bài
☐ Schema markup (nếu có)
```

## PHẦN 6: TRACKING & ANALYTICS

### 6.1 Events cần track

```javascript
// Page Views
trackView('/') // Trang chủ
trackView('/edt-core') // Trang sản phẩm
trackView('/news/1') // Chi tiết tin

// Conversions
trackEvent('cta_click', 'hero_explore')
trackEvent('modal_open', 'assessment')
trackEvent('lead_submit', 'contact_form')
trackEvent('download', 'toolkit')

// Engagement
trackEvent('chat_start', 'ai_widget')
trackEvent('search', 'library')
trackEvent('video_play', 'tutorial')
```

### 6.2 KPIs quan trọng

| KPI | Mục tiêu | Đo lường |
|-----|----------|----------|
| Organic Traffic | +20%/tháng | Google Analytics |
| Lead Generation | 50 leads/tháng | Form submissions |
| Conversion Rate | 3-5% | Leads / Visitors |
| Bounce Rate | <50% | Google Analytics |
| Time on Site | >3 phút | Google Analytics |
| Pages/Session | >3 trang | Google Analytics |

### 6.3 Báo cáo định kỳ

**Hàng tuần:**
- Traffic overview
- Top pages
- New leads

**Hàng tháng:**
- SEO rankings
- Conversion funnel
