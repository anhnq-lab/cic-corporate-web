## 1. TỔNG QUAN NỀN TẢNG

### 1.1 Giới thiệu
**CIC Platform** là hệ điều hành số toàn diện dành cho ngành xây dựng Việt Nam, giúp Chủ đầu tư, Ban QLDA, Nhà thầu và Đơn vị tư vấn:
- Chuẩn hóa quy trình theo BIM 2026
- Đảm bảo tuân thủ pháp lý (NĐ 175/2024, NĐ 111/2024)
- Tối ưu hóa hiệu suất dự án

### 1.2 Tầm nhìn
Trở thành nền tảng quản trị tiêu chuẩn (Standard Platform) kết nối CĐT - PMU - Tư vấn - Nhà thầu trong hệ sinh thái số ngành xây dựng.

## 4. ĐẶC TÍNH KỸ THUẬT

### 4.1 Công nghệ Frontend
| Công nghệ | Phiên bản | Mục đích |
|-----------|-----------|----------|
| React | 18+ | Core framework |
| TypeScript | 5+ | Type safety |
| Vite | 5+ | Build tool |
| React Router | 6+ | Routing |
| TailwindCSS | 3+ | Styling |
| React Helmet Async | - | SEO management |

### 4.2 Công nghệ Backend
| Công nghệ | Mục đích |
|-----------|----------|
| Node.js + Express | API Server |
| SQLite | Database |
| Google Gemini AI | AI Chatbot |

### 4.3 Tính năng SEO
- **Meta Tags**: Title, description, keywords
- **Open Graph**: Facebook, LinkedIn sharing
- **Twitter Cards**: Twitter preview
- **Canonical URLs**: Tránh duplicate content
- **Sitemap.xml**: Indexing cho search engines
- **Robots.txt**: Crawler control

### 4.4 Tính năng UI/UX
| Tính năng | Mô tả |
|-----------|-------|
| Responsive Design | Mobile-first, hiển thị tốt trên mọi thiết bị |
| Scroll Animation | Hiệu ứng fade-in khi cuộn |
| Error Boundary | Xử lý lỗi graceful |
| Toast Notifications | Thông báo người dùng |
| Loading States | Skeleton và spinner |

### 4.5 Tính năng bảo mật
- HTTPS enforcement
- API key protection
- Input validation
- XSS prevention

### 4.6 Analytics & Tracking
- Page view tracking
- Event tracking
- Lead capture analytics
- User behavior analysis
